# TUI pdf viewer

**Link**: [itsjunetime/tdf: A tui-based PDF viewer](https://github.com/itsjunetime/tdf)

## Setup

1. Ensure dependencies exist

2. Install via cargo

```bash
# features are optional if you want to support those filetypes
cargo install --git https://github.com/itsjunetime/tdf.git --features cbz,epub
```
